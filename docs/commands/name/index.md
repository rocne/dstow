# dstow name

<!-- dstow:short -->
Encode and decode name coordinate segments
<!-- /dstow:short -->

<!-- dstow:long -->
Encode and decode the segments of a qualified name — dstow's percent-encoding
for the characters the naming grammar reserves.

A hidden utility group: it reads and writes nothing, and exists so a script can
spell a segment exactly the way dstow does.
<!-- /dstow:long -->

## Topics

- `encode` — percent-encode a segment
- `decode` — percent-decode a segment
